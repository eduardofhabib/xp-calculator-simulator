# 🖥️ Windows XP Calculator Simulator

Simulação da interface do Windows XP no navegador, incluindo uma calculadora funcional com comportamento de janela (arrastar, minimizar e restaurar).

---

## 📖 Descrição

Este projeto recria elementos clássicos do Windows XP utilizando HTML, CSS e JavaScript.

A aplicação simula um ambiente desktop com barra de tarefas, janelas interativas e uma calculadora funcional, mantendo a lógica original do projeto base.

---

## 🚀 Funcionalidades

- 🖥️ Interface inspirada no Windows XP
- 🧮 Calculadora funcional (operações básicas)
- 🪟 Janela arrastável (drag & drop)
- ➖ Minimizar e restaurar janela
- 📌 Barra de tarefas interativa
- 🎨 Estilo visual retrô (cores e layout do XP)

---

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript
